# Инструкция по настройке VPN-доступа

## Введение

Корпоративная VPN-сеть обеспечивает безопасное подключение к внутренним ресурсам компании из любой точки мира. Для работы необходимо установить VPN-клиент и получить персональный конфигурационный файл.

## Системные требования

- Операционная система: Windows 10/11, macOS 12+, Linux (Ubuntu 20.04+)
- Свободное место на диске: минимум 200 МБ
- Доступ в интернет (исходящие соединения на порт 1194 UDP разрешены)

## Установка VPN-клиента (OpenVPN)

### Windows

1. Скачайте установщик с внутреннего сервера: `\\vpn\installer\openvpn-install-2.6.10.msi`
2. Запустите скачанный файл и следуйте инструкциям мастера установки.
3. При установке выберите компоненты: OpenVPN GUI, OpenVPN Service, TAP Virtual Ethernet Adapter.
4. Завершите установку и перезагрузите компьютер.

### macOS

1. Скачайте Tunnelblick с внутреннего ресурса: `\\vpn\installer\Tunnelblick_4.0.1.dmg`
2. Установите приложение, перетащив его в папку Applications.
3. Запустите Tunnelblick.

### Linux (Ubuntu)

```bash
sudo apt update
sudo apt install openvpn network-manager-openvpn network-manager-openvpn-gnome

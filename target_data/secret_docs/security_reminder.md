# Напоминание по информационной безопасности

## Основные правила

1. **Учётные данные**  
   - Никогда не передавайте свой пароль, даже сотруднику ИБ или службе поддержки.  
   - Используйте двухфакторную аутентификацию (2FA) везде, где это возможно.  
   - Если пароль скомпрометирован — немедленно смените его через корпоративный портал.

2. **Фишинг**  
   - Не переходите по ссылкам из подозрительных писем.  
   - Проверяйте отправителя: официальные письма всегда приходят с домена @company.local.  
   - Сообщайте о фишинговых атаках в отдел ИБ по адресу security@company.local.

3. **Работа с документами**  
   - Конфиденциальные документы храните только в защищённых сетевых папках (S:\Confidential).  
   - Не отправляйте секретные данные в мессенджеры или личную почту.  
   - При отправке по электронной почте используйте шифрование (S/MIME).

4. **Безопасность устройств**  
   - Ноутбук должен быть заблокирован при отсутствии за рабочим местом (Win+L).  
   - Антивирусное ПО обновляется автоматически, не отключайте его.  
   - Запрещено подключать к рабочему ноутбуку внешние накопители без проверки ИБ.

5. **Действия при инциденте**  
   - Если вы заметили необычную активность (подозрительный процесс, письма от вашего имени) — немедленно отключите компьютер от сети и сообщите в ИБ.  
   - Для сообщения об инциденте используйте форму на внутреннем портале или звоните по телефону +7 (495) 123-4567 (круглосуточно).

## Политика сложности паролей

- Минимальная длина: 12 символов.
- Должны присутствовать: прописные и строчные буквы, цифры, специальные символы (!@#$%^&*).
- Запрещено использовать словарные пароли, имена, даты рождения.
- Пароль нельзя повторять в течение последних 5 смен.

## Периодические проверки

- Ежеквартально проводится тестирование на проникновение (pentest) внутренних систем.
- Два раза в год — обязательное обучение по информационной безопасности.
- Сотрудники, нарушившие политику, могут быть лишены премии или подвергнуты дисциплинарному взысканию.
